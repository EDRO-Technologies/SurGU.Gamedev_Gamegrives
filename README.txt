Для добавления файлов в репозитории, воспользуйтесь рядом следующих команд:

1) git init (инициализация git папки)
2) git add . (файл, который отправляется на github, точка обозначает отправку всех файлов в данной папке)
3) git commit -m 'first commit' (добавляет комментарий на ваш пуш кода)
4) git branch -M main (выбор ветки для пуша)
5) git remote add origin (ссылка) (собственно куда вы собираетесь пушить данные, ссылка на репозиторий)
6) git push -u origin main (сам пуш данных)
